Concepts I used to do this assignment

For this assignment, I used the GitHub concepts we learned. These concepts helped me manage my project and work efficiently. The main concepts I used are listed below:

  • Repository: A repository is a place where my project files are stored on GitHub.
  • Cloning: Cloning allows me to copy the repository from GitHub to my local computer.
  • Branching: Creating a branch helps me work on changes without affecting the main project.
  • Committing: A commit saves the changes I make with a message explaining what was done.
  • Pushing: Pushing uploads my local changes to the GitHub repository.
  • Collaboration: GitHub makes it easy to collaborate by sharing code and tracking changes.

These concepts helped me successfully complete and manage this assignment using GitHub.
